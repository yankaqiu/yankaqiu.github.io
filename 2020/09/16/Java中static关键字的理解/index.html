Java中static关键字的理解

  • static含义 static表示“静态的”，被static修饰的元素被称为“静态的“，都是类级别的特征，和具体对象无关。

  • static用途

    1. 修饰静态变量。（static绝对不可以修饰局部变量！）当不同对象的该变量相同时，可将该变量用static修饰，即定义为静态变量。

      • 静态变量与实例变量区别：静态变量存储在方法区内存中，在类加载时初始化（不需要创建对象，内存就开辟了）。可直接用”类名.“的方式访问。实例变量存储在栈内存中，一个Java对象有一份，属于Java对象，不同对象值不一定相同。只能通过对象才可以访问。

    2. 修饰静态方法。静态方法是属于类的，如main方法。可在没有创建对象的情况下调用。可用”类名.“的方式访问（也可采用引用的方式访问，但即使采用引用的方式，实际上执行的时候也和引用指向的对象无关）。

      • 在静态方法内部不能直接访问实例变量和实例方法（必须通过具体的对象），反过来是可以滴。 

        public class Test01 {

    public static void name1(String name1) {
        name2();//编译报错，因未创建对象直接访问
        System.out.println(name1);
    }
    public void name2(String name2){
        name1("啦啦啦");
        System.out.println(name2);
    }

  1. 定义”静态代码块“。

    • ”静态代码块“语法格式：

      static{
      java语句
}

      在类加载时执行，并只执行一次，在一个类中可编写多个，自上而下依次执行。常用来完成预备工作，数据的准备工作。
